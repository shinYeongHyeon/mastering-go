## About Go
* 2009년 발표된 오픈 소스 프로그래밍 언어 ([https://go.dev](https://go.dev))
  * 2007년 개발 시작, 2009년 11월 10일 공개
  * [Playground](https://go.dev/play/)
* Go는 정적언어이다. 컴파일러가 있다는 뜻이다.
* 강타입 언어이다.
* 가비지 컬렉터가 존재한다.
